# RANDOM DOMAIN SCANNER

This programm generate random domains and search good sites.

## Possibilities
- Generation random domains
- Check domains for functionality
- Filter sale domains(you could tune this)
- Save results in files(kruto.txt - good, nekruto.txt - bad)
- You realy can tune this

## Files
- Main.py - main scanner 
- Clear.py - automatick clear the files
- kruto.txt - good domains
- nekruto.txt - bad domains
- bukvi.txt - report

## Controls
- Q - small report while programm is working
- Space - stop programm

## Attention!!
- You should clean the files with Clear.py or manually after programm stop, for the program to work correctly!!
